# DB Lab 1 – SQL Assignment

## Overview

This laboratory exercise is designed to familiarize students with fundamental SQL operations. It focuses on both Data Definition Language (DDL) and Data Manipulation Language (DML) commands, including creating database objects, inserting data, modifying existing records, and removing data from tables.

## Repository Contents

Question.pdf – Contains the set of problems issued by the instructor for Lab 1.

Assignment_01_Solution.sql – Includes the complete SQL script that answers all the given questions.

README.md – Provides an explanation of the assignment and instructions for execution.


## Execution Steps

1. Launch Oracle SQL Developer.


2. Create or select an appropriate database connection.


3. Load the file Assignment_01_Solution.sql into the SQL worksheet.


4. Execute the SQL commands sequentially to observe the results.



## Notes and Assumptions

All queries are written using Oracle SQL conventions.

The target database environment supports operations such as column renaming.
